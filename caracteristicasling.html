Características da linguagem

Linguagem Javascript é interpretada. Suporte a orientaçãoa objetos baseada emprotótipos, não possuindo classes nas versões iniciais(introduzido na versão ECMAScript 2015). Linguagem possui tipagem fracae dinâmica. Funções de primeira classe. A linguagem não apresenta suporte a programação multi-thread. Está atualmente na versão ECMAScript 9
